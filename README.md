# MazeFlow

MazeFlow is an interactive maze generator and solver built using Pygame. It allows users to create custom mazes and visualize different solving algorithms.

## Features
- Random maze generation
- Multiple solving algorithms:
  - Depth-First Search (DFS)
  - Breadth-First Search (BFS)
  - Dead-End Filling
- Leaderboard system (stored in JSON)
- Simple interactive UI

## Controls
- `D` → Solve using DFS  
- `B` → Solve using BFS  
- `F` → Solve using Dead-End Filling  
- Mouse → Navigate menus  

## Tech Stack
- Python
- Pygame
